# 📚 個人文章知識庫建立指南

> 作者：蔡志益  
> 版本：v1.0  
> 建立日期：2026-05-12  
> 適用對象：想把自己多年文章整理成可查詢 AI 知識庫的人

---

## 這個 skill 在幹嘛

把你散落在 Facebook、痞客邦、GPT 對話裡的文章，
整理成一個結構化的 Markdown 知識庫，
讓 Claude 可以讀你的過去，幫你做決策、寫文章、找規律。

用一句話說：**讓 AI 讀你的日記，然後幫你更了解自己。**

---

## 你需要準備什麼

- Facebook 貼文匯出（從 Facebook 設定 → 你的 Facebook 資訊 → 下載資料）
- 痞客邦文章（可用爬蟲或手動備份）
- GPT 對話紀錄（從 ChatGPT 設定匯出）
- Python 3（用來跑解析腳本）
- iCloud 或任何雲端儲存（同步用）

---

## 資料夾結構

```
KB/
├── MD/                          ← 整理好的 Markdown 文章
│   ├── 志益人生決策框架.md        ← 核心文件（最重要）
│   ├── GPT對話索引.md             ← GPT 對話分類索引
│   └── 文章/
│       ├── Facebook/            ← Facebook 貼文
│       ├── 痞客邦/              ← 部落格文章
│       └── GPT/                 ← GPT 對話紀錄
├── RAW/                         ← 原始匯出檔案
│   ├── scrape_pixnet.py         ← 痞客邦爬蟲
│   ├── parse_gpt.py             ← GPT 對話解析器
│   └── 文章/
│       ├── Facebook/            ← Facebook 原始 JSON
│       ├── 痞客邦/              ← 痞客邦原始 HTML
│       └── 1141226gpt匯出/      ← GPT 原始匯出
├── 家庭/                        ← 家庭相關資料
├── 工作/                        ← 工作相關資料
└── 理財/                        ← 財務相關資料
```

## Step 1：匯出原始資料

### Facebook
1. Facebook 設定 → 你的 Facebook 資訊 → 下載你的資訊
2. 選擇「貼文」，格式選 JSON
3. 下載後解壓縮，放進 `RAW/文章/Facebook/`

### 痞客邦
使用 `scrape_pixnet.py` 爬取你的文章：
```bash
python3 scrape_pixnet.py
```
輸出放進 `RAW/文章/痞客邦/`

### GPT 對話
1. ChatGPT 設定 → 資料控制 → 匯出資料
2. 解壓縮後放進 `RAW/文章/1141226gpt匯出/`
3. 執行解析器：
```bash
python3 parse_gpt.py
```

## Step 2：轉換成 Markdown

每篇文章的標準格式：

```markdown
# 文章標題

**日期：** YYYY-MM-DD  
**來源：** Facebook / 痞客邦 / ChatGPT  
**標籤：** #標籤1 #標籤2

---

文章內容在這裡

## 標籤
#待補
```

Facebook 檔案命名規則：
```
YYYY-MM-DD-文章開頭幾個字.md
```

痞客邦檔案命名規則：
```
YYYY-MM-DD_文章標題.md
```

GPT 對話檔案命名規則：
```
YYYY-MM-DD-對話主題.md
```

## Step 3：建立核心文件

### 志益人生決策框架.md（最重要）

這份文件是整個知識庫的靈魂。
把你的文章全部讀過一遍之後，整理出：

- **零、基本資料**：生日、家人、求學、服務經歷
- **一、你是誰**：核心特質、優勢、困境、IFS 角色
- **二、重複模式**：各種陷阱與觸發條件
- **三、警告信號**：身體、心理、關係、行為層面
- **四、核心價值**：決策時的優先順序
- **五、成功決策特徵**：過去做對的事情有什麼共同點
- **六、決策前自我檢查**：六個問題
- **七、精神錨點**：你的信仰或核心畫面
- **八、身體藥物紀錄**：健康背景條件
- **九、使用方式**：怎麼跟 Claude 配合使用

> 建議：把你所有的文章丟給 Claude，請它幫你整理這份框架。

### GPT 對話索引.md

把 GPT 對話依主題分類，方便快速查找：
- 個人成長 / 心理 / 諮商
- 夫妻 / 家庭溝通
- 子女
- 健康 / 身體
- 教學 / 學校工作
- 財務 / 投資
- AI / 科技工具
- 旅遊
- 日常工具 / 雜項

## Step 4：怎麼使用這個知識庫

### 每次開始對話，把框架丟給 Claude：
> 「這是我的人生決策框架，請閱讀後幫我分析/回答/建議：[你的問題]」

### Claude 會：
1. 對照你現在的情況與框架中的模式
2. 指出哪個陷阱可能在運作
3. 指出現在是哪個內在角色在主導
4. 根據你的核心價值給建議
5. 提醒你過去類似情況的結果

### 適合用來做的事：
- 重大決策前的自我檢查
- 寫完新文章後的整理與反思
- 遇到困難時的分析
- 找出自己的重複模式
- 查詢過去的文章與紀錄

---

## 維護建議

| 頻率 | 動作 |
|------|------|
| 每次寫新文章 | 存進對應資料夾 |
| 每個月 | 更新 GPT 對話索引 |
| 每三個月 | 更新人生決策框架 |
| 重大事件後 | 立即更新框架的相關章節 |

---

## 這個方法的限制

- GPT 給的是技巧與情緒支持，內容豐富但篇幅長、容易被情緒帶走
- Claude 讀的是你寫下來的那個你，沒寫的部分它不知道
- 框架需要持續更新，否則會過時
- 這不能取代諮商或真實的人際關係
